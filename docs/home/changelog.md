# 更新日志

Auto Research 的版本变更记录。

---

## v1.2.0 (2026-05-15)

### ✨ 新增

- 新增 RSS 数据源支持
- 新增 PDF 导出格式
- CLI 新增 `sources test` 命令，用于测试数据源连接

### 🔧 改进

- 搜索去重算法升级为语义去重，准确率提升 30%
- 配置验证更友好，错误时会提示正确格式
- 优化大结果集的内存占用

### 🐛 修复

- 修复 `max_results` 参数在部分数据源不生效的问题
- 修复 Windows 下配置文件路径解析错误

---

## v1.1.0 (2026-03-20)

### ✨ 新增

- 情感分析模块 (`SentimentAnalyzer`)
- 支持自定义 Jinja2 导出模板
- 新增 `--quiet` 静默模式

### 🔧 改进

- 搜索并发数默认值从 2 提升到 4
- 缓存由简单 TTL 升级为 LRU+TTL 双层策略
- 错误信息中文化

### 🐛 修复

- 修复并发搜索时偶发的死锁问题
- 修复 `language` 参数在 Semantic Scholar 源无效的问题

---

## v1.0.0 (2026-01-10)

### 🎉 首个正式版本

- 核心搜索引擎 `ResearchEngine`
- 支持 arXiv、Semantic Scholar、Web Search 三个数据源
- 聚类分析和摘要生成
- Markdown / HTML / JSON 三种导出格式
- CLI 命令行工具
- Python API
