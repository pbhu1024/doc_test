# 功能特性

Auto Research 提供了一套完整的研究工作流工具，以下是各项功能的详细说明。

---

## 🔍 智能搜索

多源并发搜索是 Auto Research 的核心能力。系统会同时向多个数据源发起请求，并对结果进行智能去重和排序。

### 支持的数据源

| 数据源 | 类型 | 说明 |
|--------|------|------|
| 学术论文 | API | arXiv、Semantic Scholar 等学术数据库 |
| 技术文档 | Web | 官方文档、技术博客 |
| 新闻资讯 | RSS/API | 多语言新闻源 |
| 代码仓库 | API | GitHub、GitLab 开源项目 |
| 自定义源 | 插件 | 用户自定义的任意数据源 |

### 搜索策略

- **广度优先**：默认策略，先覆盖更多来源
- **深度优先**：针对单个来源深入挖掘
- **混合模式**：结合两者优势，平衡覆盖度与深度

---

## 📊 数据分析

对搜索到的原始结果进行多维度分析。

### 聚类分析

自动将相似结果归类，生成主题地图：

```python
result = engine.search("机器学习 医疗 诊断")
analysis = engine.analyze(result.items, method="cluster")
print(analysis.clusters)
# [
#   { "topic": "影像诊断", "count": 12, "confidence": 0.92 },
#   { "topic": "药物发现", "count": 8,  "confidence": 0.87 },
#   { "topic": "病历分析", "count": 5,  "confidence": 0.81 },
# ]
```

### 情感分析

识别文本中的观点倾向，适用于舆情分析场景。

### 趋势分析

追踪话题在时间维度上的热度变化。

---

## 📝 报告生成

一键将研究结果导出为专业文档。

### 支持格式

| 格式 | 适用场景 |
|------|----------|
| Markdown | 技术文档、个人笔记 |
| PDF | 正式报告、打印分发 |
| HTML | 网页发布、交互展示 |
| JSON | 程序对接、二次开发 |

### 自定义模板

使用 Jinja2 模板引擎自定义输出样式：

```yaml
# config.yaml
output:
  template: ./templates/my_report.j2
  format: markdown
  path: ./output
```

---

## 🔗 引用管理

自动化参考文献处理。

- **自动提取**：从搜索结果中识别 BibTeX、DOI、URL 等引用信息
- **格式化输出**：支持 APA、MLA、IEEE 等多种引用格式
- **去重合并**：同一文献的多次引用自动合并
